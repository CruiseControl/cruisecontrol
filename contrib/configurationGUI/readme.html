CruiseControl Configuration Client

Version 0.9

This client may be used to edit the cruisecontrol.xml file or create new one.

Initialize:

  1. Run the ant script in the main sub-directory to build the cruisecontrol.jar file.
  2. Run ant setup in this directory to copy required jars into this project.
  3. Read the Install document to build/install the deployment you wish to use.
