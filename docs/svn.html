Subversion Access

CruiseControl's SourceForge SVN repository can be checked out without a password with the following instruction set.

svn checkout https://cruisecontrol.svn.sourceforge.net/svnroot/cruisecontrol/trunk/cruisecontrol

Committers

Only project Committers can commit to the SVN repository. When prompted for a password, use your SourceForge username and password.
