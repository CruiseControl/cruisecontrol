Anonymous CVS Access

CruiseControl's SourceForge CVS repository can be checked out through anonymous (pserver) CVS with the following instruction set. When prompted for a password for anonymous, simply press the Enter key.

export CVSROOT=:pserver:anonymous@cvs.sourceforge.net:/cvsroot/cruisecontrol
cvs login
cvs -z3 co cruisecontrol

Committer CVS Access via SSH

Only project Committers can access the CVS tree via this method. SSH1 must be installed on your client machine. Substitute developername with the proper value. Enter your site password when prompted.

export CVS_RSH=ssh
export CVSROOT=:ext:developername@cvs.sourceforge.net:/cvsroot/cruisecontrol
cvs -z3 co cruisecontrol
