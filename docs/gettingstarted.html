CruiseControl Getting Started

The following links are external resources which are relevant in setting up a new installation of CruiseControl from scratch.

  • The CruiseControl Wiki has a page Getting Started With CruiseControl.
  • The book Pragmatic Project Automation by Mike Clark has a section "Putting A Build on CruiseControl" that walks through getting CC up and running. This is part of Chapter 3: Scheduled Builds available online as a PDF.
